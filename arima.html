Use Case

This research project employs ARIMA to predict the price of Copper Futures Contracts traded on the Multi Commodity Exchange, India (MCX). The ARIMA model requires 3 hyperparameters - order of AR components, order of Integration and the order of MA components. In this project, we first try to arrive at these values statistically and then take a Machine Learning approach to fine tune these hyperparameters. This project is unique because is uses a training size of 2000 observations and makes a prediction on a rolling basis. Each rolling window comprises of the most recent 2000 observations.
