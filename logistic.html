Diabetes Classification using Logistic Regression

  • Category: Research Project
  • Technologies: Python, Machine Learning, sklearn, Pandas, Matplotlib
  • Project date: January, 2023
  • Project Codebase Here

Use Case

This research project uses Logistic Regression to predict whether a person has diabetes. The project involves cleaning the data set, performing exploratory data analysis, visualizing the data to identify relationships and then fitting a Logistic Regression model on the training data. We further examine the performance of the model using ROC Curve, Area under ROC Curve (AUC), Gini Coefficient, and then generating a lift table. Finally, we examine these metrics in the testing dataset to get a complete picture of the model's performance.
