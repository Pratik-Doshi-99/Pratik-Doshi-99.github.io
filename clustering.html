Use Case

This research project seeks to improve prediction accuracy in Regression based Machine Learning models by introducing intermediate clustering. In the project, various experiments are performed to see if clustering boosts prediction accuracy. First we fit a model on the entire training set and treat it as the baseline. Then we use a subset of the variables to cluster the training set and then fit individual models in each cluster to check if those models perform better than the baseline model. In this direction we use different models like Linear Regression, Decision Trees and Random Forests.
